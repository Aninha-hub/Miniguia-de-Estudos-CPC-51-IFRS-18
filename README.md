# Miniguia-de-Estudos-CPC-51-IFRS-18

Miniguia de Estudos desenvolvido com o apoio do NotebookLM, assistente de IA do Google para organizar conteúdos, extrair percepções e estruturar conhecimento a partir de diversas fontes, servindo como um caderno de anotações inteligente para entendimento das mudanças trazidas pelas normas contábeis CPC 51/IFRS 18 que tratam da apresentação e divulgação de Demonstrações Financeiras, com foco especial na estrutura da Demonstração do Resultado (DRE).
